wfSystemInfo

Used to retrive system capabilities at runtime, works for the following architctures:

PPC PP64 i386 .. x86 AMD64/x86_64

Uses inline assembly when and where it can by making assumptions of the compiler.  Currently this supports the following compilers:

MSVC (pretty much any version, including 64 bit versions)

GCC / Or any GCC-like compiler (includes some console compilers, clang, mingw, and other variations) and newer Intel compilers like XE studio.

SUNC, most of (if not all of IBMs compilers, and older Intel compiler for C, also the standard “retargetable lcc” compiler too.)
