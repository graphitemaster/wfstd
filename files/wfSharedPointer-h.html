wfSharedPointer

template <typename T> class wfSharedPointer

Wraps a reference-counted smart pointer around a dynamically allocated object.

Remarks

The template class describes an object that uses reference counting to manage resources.  A wfSharedPointer object effectively holds a pointer to the resource that it owns or holds a wfNullPointer.  A resource can be owned by more than one wfSharedPointer object; when the last wfSharedPointer object that owns a paticular resource is destroyed, the resource is freed.
