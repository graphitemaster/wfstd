wfStackList

template <typename T> struct wfStackList

Standard LIFO Stack

#include "wfStackList.h"

Parameters

TThe element data type to be stored in the stack.

Remarks

This container unlike many containers in the Wayfroward standard library lacks iterators.  If iterators are required use wfSingleList (a LIFO singly linked list).
